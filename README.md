# hero-code-test

# Spec
- Using .NET Core SDK 3.1
- ReactJs (need install nodejs to run this app)

# Setup
- Make sure nodejs has been installed
- update secret file:
  - Add Hero API Key: Right click the project --> Manage User Secrets --> Follow format below:
```sh
{
  "HeroApi": {
    "Key": "API_KEY"
  }
}
```
